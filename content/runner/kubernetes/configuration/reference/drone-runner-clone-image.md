__Renamed to DRONE_IMAGE_CLONE__

Optional string value. `DRONE_IMAGE_CLONE` Overrides the Docker image used to clone a repository.

**DO NOT USE DRONE_RUNNER_CLONE_IMAGE**

```
DRONE_IMAGE_CLONE=drone/git:1
```
